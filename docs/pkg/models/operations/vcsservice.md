# VcsService

If provided, only returns repositories belonging to this version control provider.


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `VcsServiceBitbucket`  | bitbucket              |
| `VcsServiceGitlab`     | gitlab                 |
| `VcsServiceGithub`     | github                 |
| `VcsServiceSelfHosted` | self-hosted            |
